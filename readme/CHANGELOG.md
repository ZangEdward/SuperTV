# 更新日志

## [5.5.8.523] - 2026-05-17

### 新增
- 增加缓存视频详情窗口，支持查看单集下载进度。
- 增加缓存视频直接播放功能。
- 增加缓存视频导出到本地相册/存储功能。

### 优化
- 优化 M3U8 缓存逻辑，下载完成后自动使用 FFmpeg 合并为 MP4 容器，提高兼容性。
- 优化缓存管理页面，增加 5 秒轮询自动更新状态。
- 优化 TV 端 UI 交互体验。

### 修复
- 修复 FFmpeg-kit 在部分构建环境下的依赖缺失问题（强制指定版本为 6.0.1 并解决 6.0-2 找不到的问题）。
- 修复缓存列表状态不同步的问题。

## [5.5.7.523] - 2026-05-16

### 新增
- 手机端增加缓存功能初版。
- 新增缓存管理界面。
