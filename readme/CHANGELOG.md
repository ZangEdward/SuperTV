# 更新日志

## [5.5.8.523] - 2026-05-17

### 新增
- 增加缓存视频详情窗口，支持查看单集下载进度。
- 增加缓存视频直接播放功能。
- 增加缓存视频导出到本地相册/存储功能。

### 优化
- 采用 LunaTV 架构：彻底移除 ffmpeg-kit 二进制依赖，改用纯 JS/TS 流处理引擎。
- 优化 M3U8 缓存逻辑：支持并发分段下载并顺序合并，显著提升缓存速度。
- 本地播放优化：内置本地 HTTP 代理服务，支持 Range 请求，大幅提升离线视频播放的稳定性与拖动体验。
- 优化缓存管理页面，增加 5 秒轮询自动更新状态。
- 优化 TV 端 UI 交互体验。

### 修复
- 修复缓存列表状态不同步的问题。

## [5.5.7.523] - 2026-05-16

### 新增
- 手机端增加缓存功能初版。
